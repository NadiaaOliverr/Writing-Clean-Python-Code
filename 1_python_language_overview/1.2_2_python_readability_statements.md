# Python Readability Control Structures

### Python
```python
cost = 100
if cost == 10:
    """ Do something"""
elif cost == 20:
    """Do something else"""
else:
    """ Do something else"""    

```

### PHP
```php
$cost = 100;
if($cost == 10) {
    // Here
} elseif($cost == 20) {
    // Here

    // Here
}
```
