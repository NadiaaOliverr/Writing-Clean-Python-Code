# Python Readability Ternary

### Python
```python
cost = 100 if True else 0
print cost
# 100
```

### PHP
```php
<?php

$cost = (true) ? 100 : 0;

# 100
```

### Ruby
```ruby

# 100
```

### JavaScript
